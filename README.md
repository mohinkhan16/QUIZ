🧠 Quiz App (JavaScript):-

A stylish and interactive Quiz Application built using HTML, CSS, and JavaScript.
This app displays multiple-choice questions, includes a countdown timer, and automatically moves to the next question.

🚀 Features:-

📌 10 Predefined Quiz Questions
⏱️ 30-Second Timer for each question
🔘 Multiple Choice Buttons
➡️ Next Button Navigation
⚡ Auto-next when time ends
🎨 Modern UI with glowing effects
📱 Responsive Design
📂 Project Structure

🔁 Flow of App:-

showQuestion() runs on load
Displays current question & options
Starts 30-second timer
User selects an option OR timer ends
Moves to next question
After last question → Quiz ends
